A hilltop node briefly waits before re-transmitting; if it hears another repeater relay the same packet with better SNR, it drops its copy instead of duplicating local traffic on airtime that's better spent on long-haul packets.

Radio

coding rate

CR is the one radio parameter that may differ between nodes — receivers decode any CR. Higher = more redundancy, better survival through noise, but slower and more channel load. Well-connected repeater: consider 5. Frequency, bandwidth, and SF must match the mesh and are fixed here.

path.hash.mode

1-byte hashes collide quickly as repeater count grows; 2-byte (65 536 values) makes collisions practically impossible at this mesh's scale and makes network debugging far easier. 3-byte only matters at millions of nodes.

Regions

firmware
region default

Default scope tags all self-originated traffic (adverts, DMs without a known path) so it can travel through repeaters that cap unscoped flood. off leaves that traffic untagged — fine while few repeaters filter, but it won't cross one that caps unscoped flood. Country-level no keeps a DM between two people anywhere in Norway routing; scope down to the city only if you're sure all your traffic is local. Needs firmware 1.15+.
